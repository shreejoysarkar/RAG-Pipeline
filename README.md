# RAG-Pipeline

This repository documents my hands-on learning and experiments with **Retrieval-Augmented Generation (RAG)** architectures.

The goal of this project is to understand how modern AI systems combine **document retrieval** with **large language models** to generate more accurate, context-aware responses.

## What I'm Exploring
- Loading and processing documents (PDFs, text)
- Generating embeddings
- Vector databases for semantic search
- Building a simple FAISS-based retrieval pipeline
- Testing different embedding approaches (local models, HuggingFace, Ollama)
- End-to-end RAG workflows

- 
## Purpose
This repository serves as a **learning sandbox** where I implement and experiment with different components of a RAG system step by step.
